The Tournament Maker

After completing the Learn React course on Codecademy, John came up with this idea to serve as his first front-end React.js project. The focus for the project is almost entirely on using Javascript and the React.js library, with less attention on HTML and CSS. With this website, users can create their own custom tournament brackets with up to 32 participants. Users can progress through the bracket by clicking the winning participant in each round, and eventually, a winner will be declared in the final round. Upon completion of the project, John deployed the website via GitHub pages.

Technologies Used

  • JavaScript/React.js
